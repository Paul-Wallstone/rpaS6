Объект_1 - Настольная лампа
  Состояние: 
     вкл./выкл.
     цвет.
  Поведение:
     включить - при включении лампа включается.
     выключить - при выключении лампа выключается.
     
Объект_2 -  штора
  Состояние:
     открыта/закрыта
     свет
  Поведение:
     открыть - при открывании, штора открывается.
     закрыть - при закрывании, штора закрывается.     

Объект_3 - кот
  Состояние:
     окрас
     размер
     уровень сытости
     настроение
  Поведение:
     кормить - при кормлении увеличивается уровень сытости.
     играть - при игрании увеличивается настроение и уменьшается уровень сытости.

Объект_4 - аккумулятор
  Состояние:
     уровень заряда
     размер
  Поведение:
     заряжать - при зарядке увеличивается уровень заряда.
     разряжать - при разрядке уменьшается уровень заряда.
     
Объект_5 - автомобильное сиденье
  Состояние:
     расстояние от руля до сиденья
     угол наклона спинки
  Поведение:
     двигать вперёд - при двиганье сиденья вперёд, расстояние от руля до сиденья уменьшается.
     двигать назад - при двиганье сиденья назад, расстояние от руля до сиденья увеличивается.
     поднимать спинку - при поднимании спинки, угол наклона спинки уменьшается.
     опускать спинку - при опускании спинки, угол наклона спинки увеличивается.
     
Объект_6 - свеча
  Состояние:
     длина
     горит/не горит
  Поведеие:
     зажечь - при зажигании свечи состояние "не горит" изменяется на "горит" и длина уменьшается каждые n секунд на 1.
     потушить - при тушении свечи состояние "горит" изменяется на "не горит" и уменьшение длины останавливается.
     
Объект_7 - холодильник
  Состояние:
     вкл./выкл.
     температура
     количество продуктов
  Поведение:
     открыть дверь - при открытии двери, температура повышается.
     закрыть дверь - при закрытии двери, температура понижается.
     полодить продукт - при помещении продукта в холодильник, количество продуктов увеличивается.
     извлечь продукт - при извлечении продукта из холодильника, колчество продуктов уменьшается.
     
Объект_8 - кофеварка
   Состояние:
      вкл./выкл.
      количество кофе
      количество воды
  Поведение:
     включить - при включении, если есть вода и кофе, кофеварка уменьшает количество кофе и воды.
     выключить - при выключении, кофеварка перестаёт варить кофе.
     засыпать кофе - при засыпании, в кофеварке увеличевается количество кофе.
     залить воду - при заливании воды, в кофеварке увеличивается количество воды.
     
Объект_9 - шаурма
   Состояние:
      температура
      количество
   Поведение:
      подогреть - при подогревании, температура повышается.
      есть - при поедании, количество уменьшается.
      
Объект_10 - книга
  Состояние:
     название
     количество страниц
     текущая страница
  Поведение:
     перелистнуть страницу вперёд - при перелистывании страницы вперёд, текущая страница увеличивается на 1.
     перелистнуть страницу назад - при перелистывании страницы назад, текущая страница уменьшается на 1.
